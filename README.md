# Plyora — Премиум изделия из фанеры

Чистая и короткая версия сайта Plyora.

**Особенности этой версии:**
- Сразу видна галерея продуктов
- Кнопка «Добавить в заказ» мгновенно добавляет изделие в форму
- Можно загрузить свои фото / чертежи / примеры
- Форма заказа = форма контактов
- Минималистичный дизайн без лишнего
- Полностью на русском языке
- Адаптивный под мобильные устройства

## Как развернуть на GitHub Pages (plyora.github.io)

1. Создайте новый репозиторий на GitHub с названием `plyora`
2. Загрузите все файлы из этой папки в репозиторий (или используйте Git push)
3. В настройках репозитория → Pages → Source: Deploy from a branch → Branch: `main` (или `master`)
4. Сайт будет доступен по адресу: `https://ваш-username.github.io/plyora`

Если хотите домен `plyora.github.io` — назовите репозиторий `plyora.github.io` и включите Pages.

## Файлы
- `index.html` — главная страница
- `images/` — фотографии изделий
- `README.md` — эта инструкция
